FDS Usage Notes

  • Running a serial FDS case
    qfds.sh path_to_fds casename.fds (user specified FDS)
    qfds.sh -r casename.fds (repository FDS)

  • Running a parallel FDS case
    qfds.sh -p nprocs path_to_fds casename.fds (user specified FDS)
    qfds.sh -r -p nprocs casename.fds (repository FDS)

  • Examining run queues
    qstat -a (all jobs)
    qstat -u user (all jobs by user)
